# Cómo reportarnos un error o incidente

Gracias por ayudarnos a mantener la plataforma funcionando bien. Cuando algo no
funcione como esperabas, **un buen reporte nos permite reproducir el problema y
arreglarlo mucho más rápido**. Sin detalles, a veces no logramos ver el mismo
error que tú viste, y el ida y vuelta retrasa la solución.

La regla de oro: **un solo mensaje por incidente, con todos los datos de abajo.**

---

## Plantilla — cópiala y complétala

```
1. Título corto del problema:
   (ej.: "No puedo enviar el formulario de contacto")

2. URL donde empezó (de dónde venías):
   (copia y pega la dirección completa de la barra del navegador)

3. URL donde se vio el problema:
   (puede ser la misma de arriba — pégala igual)

4. Qué hiciste, paso a paso:
   1)
   2)
   3)

5. Qué pasó (comportamiento actual):
   (describe el error y, si puedes, copia el mensaje EXACTO o adjunta una captura de pantalla)

6. Qué esperabas que pasara (comportamiento esperado):

7. Dispositivo y navegador:
   (ej.: iPhone con Safari / PC con Chrome / lo abrí desde Instagram o Facebook)

8. Fecha y hora aproximada en que ocurrió:
```

---

## Consejos para cada campo

- **Título corto (1):** una frase. Sirve para identificar el incidente de un vistazo.
- **URL de origen (2) y URL donde se vio (3):** lo más útil que puedes darnos.
  Copia la dirección completa desde la barra del navegador. Muchas veces el
  problema empieza en una página y se manifiesta en otra; por eso pedimos ambas.
  Si fue en la misma página, repite la misma dirección.
- **Pasos (4):** numéralos. "Entré a Contacto → llené el formulario → toqué Enviar".
- **Comportamiento actual (5):** **la captura de pantalla es oro.** Si aparece un
  mensaje de error, cópialo tal cual (mejor el texto exacto que un resumen).
- **Comportamiento esperado (6):** qué creías que debía suceder. Nos ayuda a
  entender el impacto real.
- **Dispositivo y navegador (7):** importa más de lo que parece. Por ejemplo, los
  navegadores **internos de Instagram, Facebook o LinkedIn** (cuando abres un
  enlace dentro de la app) a veces muestran errores de carga que no aparecen en
  Chrome o Safari normales. Dinos desde dónde lo abriste.
- **Fecha y hora (8):** aunque sea aproximada. Nos permite cruzar tu reporte con
  los registros del sistema y encontrar la causa exacta.

---

## Ejemplo de un buen reporte

```
1. Título: No se envía el formulario de contacto

2. URL de origen: https://xpandia.global/es/services/language-assurance

3. URL donde se vio: https://xpandia.global/es/contact

4. Pasos:
   1) Entré desde la página de Language Assurance y toqué "Hablar con un experto"
   2) Llené nombre, email, empresa y mensaje
   3) Toqué "Enviar solicitud"

5. Comportamiento actual: salió el mensaje "No se puede cargar esta página.
   Recarga para intentarlo de nuevo o regresa." (adjunto captura)

6. Comportamiento esperado: que apareciera la confirmación "Solicitud recibida".

7. Dispositivo y navegador: iPhone, abriéndolo desde Instagram.

8. Fecha y hora: 27/06/2026, alrededor de las 3:40 p.m.
```

Con un reporte así, podemos reproducirlo y resolverlo el mismo día. ¡Gracias!
